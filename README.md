📌 What This Project Is

This project is an AI-powered SEO Content Cluster Generator designed for digital marketing agencies and local businesses that want to improve organic Google rankings using structured blog ecosystems.

Instead of publishing random standalone blogs, this system creates:

SEO pillar pages
supporting cluster blogs
keyword intent mapping
local SEO optimization
internal linking strategies

The workflow follows modern topical authority and search intent frameworks used by high-performing SEO agencies.
